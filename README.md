# bitwarden-export
Export Bitwarden Vault

## What it does?
Export Bitwarden vault to your Dropbox

## Action secrets
Assign secrets in Settings -> Secrets and variables -> Actions -> New repository secret

| Name                    |
| ----------------------- |
| `BW_SERVER`             |
| `BW_CLIENTID`           |
| `BW_CLIENTSECRET`       |
| `BW_PASSWORD`           |
| `DROPBOX_APP_KEY`       | 
| `DROPBOX_APP_SECRET`    |
| `DROPBOX_REFRESH_TOKEN` |
